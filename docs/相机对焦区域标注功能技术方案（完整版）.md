# 📸 相机对焦区域标注功能技术方案（完整版）

---

# 🎯 一、目标定义（最终确认版）

实现一个**纯元数据驱动的对焦区域标注功能**：

## ✅ 需要做

- 读取相机图片中的 AF（Auto Focus）元数据
- 在图片上绘制真实对焦区域：
  - 对焦点（AF Point）
  - 对焦区域（Area）
  - 人脸（Face）
  - 眼睛（Eye）

## ❌ 明确不做

- AI 推测 ❌
- 清晰度判断 ❌
- 自动筛选 ❌
- 手机图片 ❌
- 非原始图片 ❌

---

🧩 四、核心数据结构（统一模型）
FocusRegion
struct FocusRegion {
    x: f32,   // 0~1
    y: f32,
    w: f32,
    h: f32,
    kind: FocusKind,
}
FocusKind
enum FocusKind {

    Face,      // 人脸

}
🎯 设计原则

🧩 五、Adapter 层设计（核心）
抽象接口
trait FocusAdapter {
    fn supports(metadata: &Json) -> bool;
    fn extract(metadata: &Json) -> Vec<FocusRegion>;
}
实现结构
SonyAdapter
CanonAdapter
NikonAdapter
FallbackAdapter
调度逻辑
fn get_focus_regions(meta: &Json) -> Vec<FocusRegion> {
    if SonyAdapter::supports(meta) {
        SonyAdapter::extract(meta)
    } else if CanonAdapter::supports(meta) {
        CanonAdapter::extract(meta)
    } else if NikonAdapter::supports(meta) {
        NikonAdapter::extract(meta)
    } else {
        vec![]
    }
}
