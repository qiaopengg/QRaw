## 📊 功能模块详解

### 1. 值域归一化（防止图片变黑）

**实现位置**：`app_fixed.py` - `_normalize_array_to_255()`

**功能**：

- 自动检测图像值域（[0, 1], [-1, 1], [0, 255]）
- 统一归一化到 [0, 255] float32
- 添加异常检测（图像过暗警告）

**关键代码**：

```python
def _normalize_array_to_255(arr: np.ndarray, source: str) -> np.ndarray:
    if arr_max <= 1.0 and arr_min >= 0.0:
        arr = arr * 255.0  # [0, 1] → [0, 255]
    elif arr_min < 0.0 and arr_max <= 1.0:
        arr = (arr + 1.0) * 127.5  # [-1, 1] → [0, 255]
    # ...
    return np.clip(arr, 0.0, 255.0)
```

### 2. 色彩对齐系统

**实现位置**：`color_alignment.py`

**功能模块**：

#### 2.1 Luminance-aware Mapping

```python
luminance_aware_mapping(

    strength=0.3,  # 文档建议低权重

```

- 在 LAB 色彩空间混合亮度
- 保持 AI 的色彩风格
- 恢复原图的亮度分布
- 高光和阴影区域增强保护

#### 2.2 Tone Curve 微调

```python

```

- 预设曲线：S 曲线、对比度、提亮、压暗
- 支持自定义曲线点
- 可调节强度

#### 2.3 动态范围保护

```python
preserve_dynamic_range(
    ai_result,

    preserve_ratio=0.3,

```

- 保持原图的动态范围特性
- 避免 AI 结果过度压缩或扩展

### 3. RAW 融合策略

**实现位置**：`raw_fusion.py`

**功能模块**：

#### 3.1 Tone Curve 提取

```python
extract_tone_curve_from_adjustments(adjustments)
```

- 从调整参数提取 tone curve
- 支持：exposure, contrast, highlights, shadows, whites, blacks

#### 3.2 RAW 融合

```python
blend_with_raw_tone_curve(
    ai_result,
    raw_processed,
    blend_strength=0.5,
    mode="luminance",  # luminance, color, full, adaptive
)
```

- **luminance 模式**：仅融合亮度，保持 AI 色彩
- **color 模式**：仅融合色彩，保持 AI 亮度
- **full 模式**：全通道融合
- **adaptive 模式**：自适应融合（高光和阴影区域增强）

#### 3.3 高光细节保护

```python
preserve_highlight_details(
    ai_result,
    original_img,
    threshold=200.0,
    strength=0.6,
)
```

- 在高光区域（> threshold）混合原图
- 避免过曝和细节丢失

#### 3.4 阴影层次保护

```python
preserve_shadow_details(
    ai_result,
    original_img,
    threshold=50.0,
    strength=0.5,
)
```

- 在阴影区域（< threshold）混合原图
- 避免死黑和层次丢失

### 常见问题

#### 问题 1: 图像仍然变黑

**排查**：

1. 检查 `[DEBUG] Input range` 日志
2. 确认值域归一化是否正确执行
3. 检查 Pipeline 输出类型

**解决**：

```python
# 在 _run_single() 末尾添加强制检查
if arr.mean() < 10.0:
    print(f"[ERROR] Pipeline output too dark: {arr.mean()}")
    # 强制归一化
    arr = (arr - arr.min()) / (arr.max() - arr.min() + 1e-8) * 255.0
```

#### 问题 2: 色彩对齐效果不明显

**调整参数**：

```python
# 增强亮度映射
luminanceStrength: 0.5  # 默认 0.3

# 增强曲线调整
toneCurveStrength: 0.7  # 默认 0.5

# 增强动态范围保护
dynamicRangePreserve: 0.5  # 默认 0.3
```

#### 问题 3: RAW 融合过度

**调整参数**：

```python
# 降低融合强度
rawBlendStrength: 0.3  # 默认 0.5

# 使用 luminance 模式（最保守）
rawBlendMode: "luminance"  # 默认 "luminance"
```

## 📈 性能优化

### 推理时间

| 配置              | 1024x1024 | 2048x2048 | 4096x4096 |
| ----------------- | --------- | --------- | --------- |
| Base Only         | ~30s      | ~2min     | ~8min     |
| Base + Refiner    | ~45s      | ~3min     | ~12min    |
| + Color Alignment | +2s       | +5s       | +15s      |
| + RAW Fusion      | +1s       | +3s       | +10s      |

### 显存占用

| 配置         | FP16   | FP32  |
| ------------ | ------ | ----- |
| Base Model   | ~6GB   | ~12GB |
| + Refiner    | ~8GB   | ~16GB |
| + IP-Adapter | +0.5GB | +1GB  |

### 优化建议

1. **使用 FP16**（已默认启用）
2. **启用 VAE Tiling**（已默认启用）
3. **启用 Attention Slicing**（已默认启用）
4. **调整 Tile Size**：
   - 显存充足：1024
   - 显存紧张：768 或 512
5. **禁用 Refiner**（如果不需要极致质量）
