## 📊 完整数据流

```
┌─────────────────────────────────────────────────────────────────────┐
│                         用户操作                                     │
│                    点击"风格迁移"按钮                                │
└─────────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────────┐
│                    前端 (ChatPanel.tsx)                              │
│  1. 调用 Tauri Command: run_style_transfer()                        │
│  2. 监听事件: 'style-transfer-stream'                               │
│  3. 在 ThinkingBlock 中显示进度                                     │
└─────────────────────────────────────────────────────────────────────┘
                              │
                              ▼
┌─────────────────────────────────────────────────────────────────────┐
│              Rust 后端 (style_transfer_runtime.rs)                   │
│                                                                      │
│  run_style_transfer()                                                │
│  ├─ 1. 检查服务状态                                                 │
│  ├─ 2. 调用 invoke_python_style_transfer()                          │
│  │    ├─ a. 生成 UUID task_id                                       │
│  │    ├─ b. 启动 SSE 监听任务 (tokio::spawn)                        │
│  │    │    └─ listen_sse_progress()                                 │
│  │    │         ├─ 连接 GET /v1/style-transfer/progress/{task_id}  │
│  │    │         ├─ 解析 SSE 数据流                                  │
│  │    │         └─ 转发到前端: emit_style_transfer_status()         │
│  │    └─ c. 发起 POST /v1/style-transfer (带 task_id)              │
│  └─ 3. 返回结果                                                     │
└─────────────────────────────────────────────────────────────────────┘
                    │                          │
                    │ HTTP POST                │ SSE Stream
                    │ (异步)                   │ (实时)
                    ▼                          ▼
┌─────────────────────────────────────────────────────────────────────┐
│                  Python 服务 (app.py)                                │
│                                                                      │
│  POST /v1/style-transfer                                             │
│  ├─ 1. 接收请求 (含 task_id)                                        │
│  ├─ 2. 执行风格迁移                                                 │
│  │    ├─ _print_progress(0%, "开始...", task_id)                   │
│  │    ├─ _print_progress(5%, "加载图像...", task_id)               │
│  │    ├─ _run_tiled() / _run_single()                               │
│  │    │    ├─ _print_progress(10%, "准备控制图像...", task_id)     │
│  │    │    ├─ _print_progress(20%, "运行 SDXL...", task_id)        │
│  │    │    ├─ ... (每个 tile 更新进度)                             │
│  │    │    └─ _print_progress(90%, "推理完成", task_id)            │
│  │    ├─ _print_progress(92%, "色彩对齐...", task_id)              │
│  │    ├─ _print_progress(94%, "RAW 融合...", task_id)              │
│  │    ├─ _print_progress(96%, "保存文件...", task_id)              │
│  │    └─ _print_progress(100%, "完成！", task_id)                  │
│  └─ 3. 返回结果                                                     │
│                                                                      │
│  GET /v1/style-transfer/progress/{task_id}                           │
│  ├─ 1. 创建进度队列                                                 │
│  ├─ 2. 返回 SSE 流                                                  │
│  │    └─ progress_stream()                                          │
│  │         ├─ 从队列读取进度数据                                    │
│  │         ├─ 格式化为 SSE: "data: {...}\n\n"                       │
│  │         ├─ 发送心跳: ": heartbeat\n\n"                           │
│  │         └─ 检测完成信号: type="done"                             │
│  └─ 3. 清理队列                                                     │
└─────────────────────────────────────────────────────────────────────┘
```

```
    │  {percentage: 10,          │                            │

    │                            │                            │
```

## 📡 SSE 数据格式

### 进度消息

```
data: {"percentage": 50, "description": "运行 SDXL...", "bar": "=========================                         ", "message": "[PROGRESS] 迁移进度：========================= (50%) - 运行 SDXL..."}

```

### 完成消息

```
data: {"type": "done", "percentage": 100, "output_image_path": "/tmp/output.tiff", "preview_image_path": "/tmp/preview.png"}

```

### 心跳消息

```
: heartbeat

```

## 🔧 故障排查

### 问题 1: 看不到进度条

**可能原因：**

- Python 服务未启动
- SSE 连接失败
- task_id 未正确传递

**解决方法：**

```bash
# 检查服务状态
curl http://127.0.0.1:7860/health

# 检查 SSE 端点
curl -N http://127.0.0.1:7860/v1/style-transfer/progress/test

# 查看 Rust 日志
# 在 listen_sse_progress() 中添加 println!
```

### 问题 2: 进度卡住不动

**可能原因：**

- AI 推理耗时长（正常）
- 队列阻塞
- SSE 连接断开

**解决方法：**

```bash
# 检查 Python 日志
export QRAW_DEBUG="1"
python3 app.py

# 查看是否有 [PROGRESS] 输出
```

### 问题 3: 进度跳跃或不连续

**可能原因：**

- 网络延迟
- 队列积压
- 进度更新过快

**解决方法：**

- 正常现象，不影响功能
- 可以添加节流（throttle）优化
