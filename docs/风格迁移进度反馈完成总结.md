## 📋 功能概述

实现了基于 Server-Sent Events (SSE) 的实时进度反馈系统，用户在执行风格迁移时可以看到详细的进度条和状态信息。

   @app.get("/v1/style-transfer/progress/{task_id}")
   async def style_transfer_progress(task_id: str):
       """SSE 端点：实时推送风格迁移进度"""
       queue = _create_progress_queue(task_id)

       def event_generator():
           try:
               for data in progress_stream(task_id, queue):
                   yield data
           finally:
               _remove_progress_queue(task_id)

       return StreamingResponse(
           event_generator(),
           media_type="text/event-stream",

               "Connection": "keep-alive",
               "X-Accel-Buffering": "no"

       )

**新增函数：**

```rust
/// 监听 SSE 进度流
async fn listen_sse_progress(
    sse_url: &str,
    app_handle: &tauri::AppHandle,
) -> Result<(), String> {
    // 1. 创建 HTTP 客户端
    let client = Client::builder()
        .timeout(Duration::from_secs(3600))
        .build()?;

    // 2. 连接 SSE 端点
    let response = client.get(sse_url).send().await?;

    // 3. 获取字节流
    let mut stream = response.bytes_stream();
    let mut buffer = String::new();

    // 4. 逐行解析 SSE 数据
    while let Some(chunk) = stream.next().await {
        let chunk = chunk?;
        let text = String::from_utf8_lossy(&chunk);
        buffer.push_str(&text);

        // 按行处理
        while let Some(line_end) = buffer.find('\n') {
            let line = buffer[..line_end].trim().to_string();
            buffer = buffer[line_end + 1..].to_string();

            // 解析 "data:" 行
            if line.starts_with("data:") {
                let json_str = line[5..].trim();

                if let Ok(progress_data) = serde_json::from_str::<Value>(json_str) {
                    // 检查完成信号
                    if progress_data.get("type").and_then(|v| v.as_str()) == Some("done") {
                        return Ok(());
                    }

                    // 提取并转发进度消息
                    if let Some(message) = progress_data.get("message").and_then(|v| v.as_str()) {
                        emit_style_transfer_status(app_handle, message);
                    }
                }
            }
        }
    }

    Ok(())
}
```

## 🔧 技术细节

### SSE 数据格式

Python 服务发送的 SSE 数据格式：

```
data: {"percentage": 50, "description": "处理中...", "bar": "=========================                         ", "message": "[PROGRESS] 迁移进度：========================= (50%) - 处理中..."}

data: {"type": "done", "percentage": 100, "output_image_path": "/path/to/output.tiff", "preview_image_path": "/path/to/preview.png"}

: heartbeat
```

### 依赖项

**Python:**

- `fastapi` - Web 框架
- `queue.Queue` - 线程安全队列
- `StreamingResponse` - SSE 响应

**Rust:**

- `reqwest` (with `stream` feature) - HTTP 客户端
- `futures_util::StreamExt` - 异步流处理
- `uuid` (with `v4` feature) - UUID 生成
- `tokio` - 异步运行时

所有依赖项已在 `Cargo.toml` 中配置，无需额外安装。
